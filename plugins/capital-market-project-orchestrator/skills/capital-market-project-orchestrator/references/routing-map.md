# 上市总控路由表

## 一句话入口

用户说“用上市总控”时，先判断任务属于哪类，不要求用户记住具体 skill 名称。

## 路由矩阵

| 用户意图 | 首选能力 | 输出 |
|---|---|---|
| 制定新三板挂牌计划 | `capital-market-project-orchestrator` | 总控计划、周计划、资料清单、中介分工 |
| 证券部怎么搭建 | `board-secretary-hr` | 岗位职责、招聘画像、培训计划、交接清单 |
| 判断未来是否适合北交所 | `bse-selection-analyzer` | 北交所适配分析、成长性/流动性/专精特新关注点 |
| 做信息披露日历或公告监控 | `disclosure-notice-monitor` | 信披事项清单、定报日历、重大事项监控 |
| 分析财务质量 | `financial-statement-analyzer` | 盈利质量、现金流、异常指标、整改建议 |
| 做同行比较 | `peer-comparison-analyzer` | 可比公司表、估值差异、业务定位 |
| 起草章程、股东会/董事会文件 | `guquan-gongsi-zhili` | 文书框架、条款清单、议案/决议草稿 |
| 做财务预测和三表模型 | `3-statements-ultra` | 三表模型结构、关键假设、校验清单 |
| 做正式 Word/Excel/PPT/PDF 文件 | document skills | 可交付办公文件与版式检查 |

## 推荐项目包

### 新三板挂牌启动包

- 项目里程碑表
- 中介机构分工表
- 尽调资料清单
- 历史沿革问题清单
- 财务规范问题清单
- 三会和治理整改清单
- 股东与实际控制人核查清单

### 北交所/创业板预评估包

- 板块路径比较
- 财务指标差距
- 行业与创新属性说明
- 同业对标表
- 规范运行年限与历史问题清单
- 募投项目与成长性论证框架

### 挂牌后持续督导包

- 年度信披日历
- 重大事项触发清单
- 董监高培训计划
- 关联交易/对外担保/资金占用监控
- 投资者沟通材料框架

## 安全边界

- 具体上市条件、财务阈值、审核口径和申报规则可能更新；涉及结论时必须查官方规则或标注待核验。
- 不替代律师、会计师、券商、评估师的专业签字意见。
- 对“能否上市”“是否构成实质障碍”只能给风险判断和整改建议，不给保证性结论。
