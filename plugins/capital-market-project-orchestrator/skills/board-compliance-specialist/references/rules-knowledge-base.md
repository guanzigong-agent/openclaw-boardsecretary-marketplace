# 法规库维护

本模块用于维护 `新三板法律法规知识库`。

## 维护范围

- `README.md`
- `00_索引/法规总索引.md`
- `00_索引/规则更新台账.md`
- `01_专题/*.md`
- `02_模板/法规笔记模板.md`

## 维护原则

- 官方来源优先。
- 规则名称、发布机关、发布日期、实施日期、时效性和官方链接必须可追溯。
- 新增、修订、废止、替代关系应进入规则更新台账。
- 专题笔记只做工作索引和执行提示，不替代专业法律意见。
- 每次维护应记录检查日期、检查人、检查范围、后续动作。

## 法规笔记字段

| 字段 | 要求 |
| --- | --- |
| 法规名称 | 与官方名称一致 |
| 发布机关 | 全国人大/国务院/证监会/全国股转系统等 |
| 规则层级 | 法律/行政法规/部门规章/规范性文件/业务规则/业务指南 |
| 发布日期 | 以官方页面为准 |
| 实施日期 | 以官方文本为准 |
| 时效性 | 现行有效/已修订/已废止/待核验 |
| 官方链接 | 官方页面或文件链接 |
| 适用场景 | 股改/挂牌/治理/信披/融资/交易/重组/终止挂牌 |
| 证券部关注点 | 证券部执行时需要关注的事项 |
| 董秘确认事项 | 需要董秘判断或决策的事项 |
| 中介确认事项 | 律师/会计师/券商/评估机构确认事项 |
