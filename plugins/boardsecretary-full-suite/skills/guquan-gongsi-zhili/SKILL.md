# 公司治理与股权事务处理指南 (guquan-gongsi-zhili)

本 Skill 旨在为处理公司治理、股权变更、增资扩股及工商登记等事务提供标准化的操作指引、法律文书模板和参考依据。所有内容均基于中国《公司法》（2023年修订版）及相关法律法规编写，确保文书的专业性、严谨性和合规性。

## When to Use

本 Skill 适用于以下场景：

- 起草股权转让协议、增资扩股协议等法律文书
- 编写股东会决议或董事会决议
- 处理工商变更登记相关事务
- 进行公司治理结构调整（变更法定代表人、董事、监事等）
- 防范和处理股权纠纷（代持股、公司僵局等）

## 一、 适用场景与核心能力

本 Skill 适用于以下常见公司治理与股权变动场景：

1.  **股权转让：** 内部股东之间的股权流转，或向外部投资人转让股权，涉及优先购买权的处理、转让协议的起草及交割安排。
2.  **增资扩股：** 公司引入新股东或原股东追加投资，涉及公司估值、增资款的分配（注册资本与资本公积）、优先认购权的处理及增资协议的起草。
3.  **公司治理结构调整：** 变更法定代表人、董事、监事或高级管理人员，修改公司章程，以及处理股东会决议的起草与表决程序。
4.  **工商变更登记：** 梳理上述变动事项在市场监督管理部门办理变更登记的标准化流程及所需材料清单。
5.  **股权纠纷防范与处理：** 针对代持股、损害公司利益、股东知情权受限及公司僵局等常见纠纷，提供风险提示与处理建议。

## 二、 目录结构说明

本 Skill 包含以下核心资源目录，请在处理具体事务时查阅相应文件：

### 1. 法律文书模板 (`{baseDir}/templates/`)

该目录包含了常用的标准化法律文书模板。在起草文书时，请务必根据实际情况替换括号 `[]` 中的内容，并根据具体交易需求增删条款。

*   **`equity_transfer_agreement.md` (股权转让协议模板):** 适用于有限责任公司的股权转让交易，包含了转让标的、价格、支付方式、交割条件、违约责任等核心条款。
*   **`capital_increase_agreement.md` (增资扩股协议模板):** 适用于公司增加注册资本的场景，明确了增资方案、股权结构变动、增资款支付及各方陈述与保证。
*   **`shareholders_resolution.md` (股东会决议模板):** 涵盖了股权转让、增资扩股、修改章程及人员变更等常见决议事项的标准化表述，并提示了法定表决权比例要求。

### 2. 参考资料 (`{baseDir}/references/`)

该目录提供了相关的法律法规速查、实务操作指南及风险防范建议，供咨询和起草文书时参考。

*   **`company_law_key_articles.md` (《公司法》关键条款速查):** 摘录了2023年修订版《公司法》中关于出资、股权转让、增资减资及公司治理机构的核心条款，重点提示了新法修订带来的重大变化（如五年实缴制、外部转让取消同意权等）。
*   **`registration_change_guide.md` (工商变更登记流程指南):** 梳理了股权转让、增资扩股及法定代表人变更在市场监督管理部门办理变更登记的标准流程、前置程序（如税务申报）及所需核心材料清单。
*   **`equity_dispute_guide.md` (常见股权纠纷处理要点):** 总结了代持股纠纷、董监高违规、股东知情权纠纷及公司僵局等四类常见纠纷的处理原则与防范建议，帮助在起草文书时提前规避风险。

## 三、 操作流程建议

在处理公司治理与股权事务时，建议遵循以下标准操作流程：

1.  **需求分析与尽职调查：** 明确交易背景、目的及各方诉求。对于复杂的股权转让或增资扩股，建议先进行必要的财务和法律尽职调查。
2.  **方案设计与条款磋商：** 根据交易目的，设计合理的股权结构、交易对价及交割路径，并与各方磋商核心商业条款。
3.  **查阅参考资料：** 仔细阅读 `{baseDir}/references/` 目录下的相关指南，确保方案设计符合最新《公司法》的规定，并评估潜在的法律风险。
4.  **起草法律文书：** 调取 `{baseDir}/templates/` 目录下的相应模板，根据磋商结果起草正式的法律文书（如股权转让协议、增资扩股协议）。
5.  **履行内部决策程序：** 协助起草股东会决议（参考 `shareholders_resolution.md`）或董事会决议，确保交易获得公司内部的合法授权。
6.  **签署与交割：** 组织各方正式签署法律文书，并监督款项的支付与标的股权的交割。
7.  **办理工商与税务变更：** 参照 `{baseDir}/references/registration_change_guide.md` 的指引，协助公司完成税务申报及工商变更登记手续，确保交易产生对抗善意第三人的法律效力。

## 四、 风险提示与注意事项

在使用本 Skill 及相关模板时，请务必注意以下事项：

*   **法律适用性：** 本 Skill 提供的模板及指南基于中国现行法律法规编写。由于法律法规可能不断修订，且各地市场监管部门的具体操作要求可能存在差异，实际应用时应结合最新规定及当地政策进行调整。
*   **个性化定制：** 模板仅提供基础框架，无法涵盖所有复杂的商业交易安排。对于涉及对赌协议、特殊表决权安排、员工股权激励等复杂条款的交易，必须根据具体情况进行深度定制，必要时应寻求专业律师的协助。
*   **新《公司法》的适用：** 请特别注意2023年修订版《公司法》带来的重大变化，例如注册资本五年实缴制、未届出资期限股权转让的责任承担、以及外部股权转让规则的调整。在起草相关条款时，务必确保符合新法要求。
*   **税务合规：** 股权转让通常涉及个人所得税或企业所得税的缴纳。在办理工商变更登记前，必须依法完成税务申报，取得完税证明。平价或低价转让股权可能面临税务机关核定征收的风险，应提前做好税务筹划。
