# 证券部文件合规关口

本模块用于建立“证券部输出文件必须经过合规检查”的流程。

## 适用文件

- 三会通知、议案、决议、会议记录、签署页和归档目录。
- 定期报告资料清单、底稿目录、部门反馈表和重大事项资料。
- 监管沟通记录、问询回复草稿和回复支撑材料。
- 信息披露底稿、公告草稿、备查文件清单。
- 股改、挂牌、股权、董监高、关联关系和中介机构协同文件。
- 证券部制度、流程、台账、对外报送材料和工作简报。

## 流转规则

1. 起草人完成文件初稿。
2. 证券事务代表完成事实、附件、版本和归档预检查。
3. 合规专员进行合规检查，输出问题清单和结论分级。
4. 起草人根据意见修改。
5. 董秘复核需决策、披露、监管或专业机构确认事项。
6. 必要时提交券商、律师、会计师或其他专业机构确认。
7. 定稿后归档合规检查记录。
