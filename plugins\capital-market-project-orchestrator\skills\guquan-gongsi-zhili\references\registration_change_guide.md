# 工商变更登记流程指南（股权转让/增资扩股/法定代表人变更）

本指南梳理了公司在发生股权变动、增资扩股及法定代表人变更等常见治理事项时，向市场监督管理部门（原工商局）申请变更登记的标准流程及所需材料。

> **提示：** 各地市场监管部门的具体要求可能略有差异，实际办理前建议登录当地政务服务网或拨打咨询电话确认最新要求。本指南以普遍适用的规则为基础。

## 一、 股权转让变更登记流程

### 1. 适用场景
有限责任公司股东之间转让股权，或股东向股东以外的人转让股权，导致股东姓名（名称）或出资比例发生变化的。

### 2. 办理流程
1.  **内部决议与协议签署：**
    *   转让方与受让方达成初步意向。
    *   （若向外部转让）转让方书面通知其他股东，其他股东在30日内答复是否行使优先购买权。
    *   其他股东放弃优先购买权（或视为放弃）后，转让方与受让方正式签署《股权转让协议》。
    *   召开股东会，形成同意股权转让及修改公司章程的《股东会决议》。
2.  **税务申报（完税证明）：**
    *   **关键前置步骤：** 股权转让必须先到税务局办理个人所得税（自然人股东）或企业所得税（法人股东）的纳税申报。
    *   税务局审核股权转让价格是否合理（平价或低价转让需有正当理由，否则税务局有权核定转让收入）。
    *   取得《股权转让所得个人所得税完税证明》或税务机关出具的相关凭证。
3.  **网上申报/预约：**
    *   登录当地“企业开办一网通办”平台或政务服务网。
    *   填报变更信息，上传电子版申请材料。
    *   系统审核通过后，预约线下提交纸质材料的时间（部分地区支持全流程电子化，无需线下提交）。
4.  **线下提交材料（如需）：**
    *   法定代表人或委托代理人携带纸质材料前往行政服务中心市场监管窗口办理。
5.  **领取新营业执照：**
    *   审核通过后，领取换发的营业执照（若涉及法定代表人、公司名称等记载事项变更）。

### 3. 所需核心材料清单
*   [ ] 《公司登记（备案）申请书》（法定代表人签字，加盖公章）。
*   [ ] 《指定代表或者共同委托代理人授权委托书》及代理人身份证复印件。
*   [ ] 变更后的《公司章程》或《章程修正案》（法定代表人签字，加盖公章）。
*   [ ] 股东会决议（全体股东签字/盖章，同意转让及修改章程）。
*   [ ] 股权转让协议（转让方与受让方签字/盖章）。
*   [ ] 新股东的主体资格证明（自然人提供身份证复印件，企业提供营业执照复印件加盖公章）。
*   [ ] 税务机关出具的完税证明或相关涉税信息凭证。
*   [ ] 营业执照正、副本原件。

## 二、 增资扩股变更登记流程

### 1. 适用场景
公司增加注册资本，引入新股东或原股东追加投资。

### 2. 办理流程
1.  **内部决议与协议签署：**
    *   公司与投资方协商确定增资方案（增资金额、认购价格、投后估值等）。
    *   召开股东会，经代表三分之二以上表决权的股东通过，形成同意增资及修改公司章程的《股东会决议》。
    *   公司、原股东与新股东签署《增资扩股协议》。
2.  **资金实缴（视认缴/实缴制而定）：**
    *   若约定实缴，投资方将增资款汇入公司指定账户。
    *   （非强制要求，但为规范起见）可由会计师事务所出具验资报告。
3.  **网上申报/预约：**
    *   登录政务服务网填报增资信息。
4.  **线下提交材料/全流程电子化办理：**
    *   提交变更申请。
5.  **领取新营业执照：**
    *   营业执照上的“注册资本”一栏将更新为增资后的金额。

### 3. 所需核心材料清单
*   [ ] 《公司登记（备案）申请书》。
*   [ ] 《指定代表或者共同委托代理人授权委托书》及代理人身份证复印件。
*   [ ] 变更后的《公司章程》或《章程修正案》。
*   [ ] 股东会决议（代表2/3以上表决权股东签字/盖章）。
*   [ ] （若引入新股东）新股东的主体资格证明。
*   [ ] 营业执照正、副本原件。
*   *注：一般不需要提交《增资扩股协议》原件，但建议备查。*

## 三、 法定代表人变更登记流程

### 1. 适用场景
公司原法定代表人辞职、被免职，或公司章程规定的担任法定代表人的职务（董事长、执行董事或经理）发生人员变更。

### 2. 办理流程
1.  **内部决议：**
    *   根据章程规定，召开股东会或董事会，作出免去原法定代表人职务、选举/聘任新法定代表人的决议。
2.  **网上申报与材料提交：**
    *   流程同上。
3.  **领取新营业执照：**
    *   营业执照上的“法定代表人”一栏将更新。
4.  **后续联动变更（重要）：**
    *   **税务变更：** 需同步在税务系统更新法定代表人信息。
    *   **银行基本户变更：** 需前往开户行更换印鉴卡，变更账户法定代表人信息。
    *   **社保/公积金变更：** 及时更新经办系统信息。
    *   **各类资质证书变更：** 如高新技术企业证书、医疗器械经营许可证等，若载有法定代表人信息，需逐一申请变更。

### 3. 所需核心材料清单
*   [ ] 《公司登记（备案）申请书》（由**新任**法定代表人签字，加盖公章）。
*   [ ] 《指定代表或者共同委托代理人授权委托书》。
*   [ ] 股东会决议或董事会决议（关于免去原职务、选举/聘任新职务的决定）。
*   [ ] 新任法定代表人任职文件（如聘书）及身份证复印件。
*   [ ] 原法定代表人的免职文件。
*   [ ] 营业执照正、副本原件。

---
**风险提示：**
1.  **“人证合一”：** 办理变更登记时，法定代表人、股东等关键人员通常需要通过政务APP进行实名认证和电子签名。
2.  **债务隔离：** 股权转让变更登记完成（公示）是产生对抗善意第三人效力的关键节点。未办理变更登记的，受让人不得对抗善意第三人。
